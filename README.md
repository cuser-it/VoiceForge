# VoxCPM2 交互式录音训练数据生成器

一个零构建、零后端依赖的本地 Web 工具，用于录制 VoxCPM2 LoRA 训练数据并生成标准数据集文件。

## 启动

```bash
python3 -m http.server 4173
```

打开：

```text
http://127.0.0.1:4173/
```

不要直接用 `file://` 打开页面；浏览器会拦截本地 JSON 文案库加载。

## 使用流程

1. 选择录制条数：20、30 或 50。
2. 可选上传 `.txt` 或单列 `.csv` 自定义文案。
3. 点击“重新洗牌”预览并更换本次文案列表。
4. 按空格开始录音，朗读后自然停顿约 3 秒。
5. 自动停止后试听，按回车保存并进入下一条。
6. 不满意时按空格重录；按 `Ctrl+Z` 撤销本次会话上一条保存记录。

## 输出

支持两种输出方式：

- Chrome/Edge 等支持目录写入的浏览器：点击“选择 output 目录”，保存时自动写入。
- 其他浏览器：点击“导出 ZIP”，下载完整数据集。

输出结构：

```text
output/
├── audio/
│   ├── sample_001.wav
│   └── ...
├── train.jsonl
└── metadata.csv
```

`train.jsonl` 示例：

```json
{"audio":"audio/sample_001.wav","text":"李奶奶请老板给我打五桶水","duration":3.52}
```

音频为 16kHz、单声道 WAV。

## 实现范围

- 内置 300 条普通话文案，位于 `data/text_pool.json`。
- 支持 TXT/CSV 自定义文案追加到内置池。
- 支持随机抽取、文案预览和重新洗牌。
- 支持实时波形、VAD 静音停止、自动回放。
- 支持本次会话撤销，不触碰历史录音文件。
- 支持 `train.jsonl`、`metadata.csv` 和 `audio/` 目录结构。
